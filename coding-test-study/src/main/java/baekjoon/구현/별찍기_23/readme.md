# 별 찍기 - 23
### 문제 

예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.

### 입력

첫째 줄에 N(2 ≤ N ≤ 100)이 주어진다.

### 출력

첫째 줄부터 차례대로 별을 출력한다.

### 예제 입력 1

~~~
2
~~~

### 예제 출력 1

~~~
** **
 ***
** **
~~~

### 예제 입력 2

~~~
5
~~~

### 예제 출력 2

~~~
*****       *****
 *   *     *   *
  *   *   *   *
   *   * *   *
    *   *   *
   *   * *   *
  *   *   *   *
 *   *     *   *
*****       *****
~~~
