# วัฑ LCS 
### นฎมฆ

LCS(Longest Common Subsequence, รึภๅ ฐ๘ล๋ บฮบะ ผ๖ฟญ)นฎมฆดย ตฮ ผ๖ฟญภฬ มึพ๎มณภป ถง, ธ๐ตฮภว บฮบะ ผ๖ฟญภฬ ตวดย ผ๖ฟญ ม฿ ฐกภๅ ฑไ ฐอภป รฃดย นฎมฆภฬดู.

ฟนธฆ ต้พ๎, "รึน้มุ"ฐ๚ "น้มุฟยถ๓ภฮ"ภว LCSดย "น้มุ"ภฬฐํ, "ฝบลธฦฎธตลฉ"ฟอ "ฝบฦฎถ๓ลไฤณฝบลอ"ภว LCSดย "ฝบฦฎ"ภฬดู.

### ภิทย

รนยฐ มูฐ๚ ตัยฐ มูฟก ตฮ นฎภฺฟญภฬ มึพ๎ม๘ดู. นฎภฺฟญภบ รึด๋ 1000ฑภฺภฬฐํ, ภฏดฯฤฺตๅ U+AC00(ฐก)บฮลอ U+D7A3(ฦR)ฑ๎ม๖ทฮธธ ภฬท็พ๎มฎ ภึภธธ็, UTF-8ทฮ ภฮฤฺต๙ ตวพ๎ ภึดู.

### รโทย

รนยฐ มูฟก ภิทยภธทฮ มึพ๎ม๘ ตฮ นฎภฺฟญภว LCSภว ฑๆภฬธฆ รโทยวัดู.
